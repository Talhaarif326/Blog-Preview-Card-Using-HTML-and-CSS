## 📚 What I Learned

- How to center a card perfectly using **Flexbox**  
- Working with `box-shadow`, `border-radius`, and spacing  
- Creating a simple **responsive-friendly layout**  

css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
